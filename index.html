How it works

We’re used to looking at astronomical data, but we can listen to it too! This is a type of image sonification, which converts the information used to make an image into sound. As the playhead scans across the image, the brightness and color data in the image is used to control the sounds you hear.


Data

This selects which data will control the musical pitches you hear. If Brightness is selected, then darker regions will produce lower notes and brighter regions will produce higher pitched notes. If Color is selected then redder light will produce lower notes and bluer light will produce higher notes.


Harmony

This controls the mood by setting which set of musical notes that are used to represent the data. Each choice corresponds to a particular musical scale.


Soundwave

This controls the type of soundwave used to hear extended regions of light such as nebulae or large galaxies. These range from smooth to harsher sounds.


Instrument

This controls the musical instrument used to hear small, compact sources of light such as stars or distant galaxies.
